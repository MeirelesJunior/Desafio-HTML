Tags H1 até H6.

H1 até H6 são usadas para definir títulos e subtítulos em HTML. Cada tag representa um nível de importância ou hierarquia.

H1 é o título principal, H2 é um subtítulo de nível inferior, e assim por diante até H6. Motores de busca e leitores de tela usam essas tags para entender a estrutura do conteúdo.

A aparência dos títulos depende do estilo CSS aplicado. Use as tags corretamente para melhorar a acessibilidade e usabilidade. Evite pular níveis na hierarquia dos títulos. Recomenda-se usar apenas um H1 por página. Uso adequado dessas tags melhora a organização e compreensão do conteúdo. (voltar)

Tag P.

A tag p é usada para criar um parágrafo de texto em uma página HTML,é uma tag de conteúdo que representa um bloco de texto.

O conteúdo dentro da tag p é exibido em uma nova linha e é separado por espaços em branco.(voltar)

Tag Mark.

A tag mark é usada para destacar um trecho de texto em uma página HTML.O objetivo principal da tag é chamar a atenção do leitor para um determinado trecho de texto. No entanto, é possível personalizar o estilo usando CSS para definir cores diferentes, fonte, tamanho, entre outros. A tag é especialmente útil quando se deseja realçar termos de pesquisa, destaques de citações ou informações importantes. É importante usar a tag com moderação e apenas para destacar informações relevantes para a compreensão do conteúdo. (voltar)

Tag Small.

Tags Ol - Ul - LI.

A tag OL é usada para criar uma lista ordenada, onde os itens são numerados ou ordenados de alguma forma. Os itens são definidos com a tag Li.
Ex:

  1. Item 1
  2. Item 2
  3. Item 3

A tag Ul é usada para criar uma lista não ordenada, onde os itens não são numerados ou ordenados. Os itens também são definidos com a tag Li.
Ex:

  • Pc
  • PS5
  • Xbox X

A tag Li é usada para definir um item em uma lista, seja ela ordenada ou não. Ela representa um único item da lista.

    Tag A

    A tag a é usada para criar um hyperlink, ou seja, um link clicável em uma página HTML. Ela permite aos usuários navegarem para outras páginas da web, seções da mesma página ou até mesmo arquivos para download. É composta por dois elementos principais: o atributo href, que define o destino do link, e o conteúdo entre as tags , que é exibido como o texto do link. O atributo href pode conter URLs, endereços de e-mail, IDs de elementos na mesma página, nomes de arquivos, entre outros. Além disso, é possível adicionar o atributo target para especificar como o link deve ser aberto, como em uma nova janela ou na mesma janela.

    Tag Sub - Sup

    A tag sub é usada para exibir texto subscrito, que é posicionado ligeiramente abaixo da linha de base normal. Ela é frequentemente usada para exibir elementos como índices, fórmulas químicas, datas e notas de rodapé. Por exemplo, a fórmula química H2O pode ser escrita como H2O, onde o número 2 é exibido em formato subscrito.

    A tag sup é usada para exibir texto sobrescrito, que é posicionado ligeiramente acima da linha de base normal. Ela é comumente usada para exibir elementos como expoentes, números ordinais, fórmulas matemáticas e notas de rodapé. O texto envolvido pela tag é reduzido em tamanho e é exibido com uma posição superior em relação ao texto circundante. Por exemplo, 23 representa o número 2 elevado ao cubo, onde o número 3 é exibido em formato sobrescrito.

    Tag Blockquote

    A tag blockquote é usada para destacar um bloco de citação em uma página HTML. Ela é usada para indicar que o conteúdo dentro dela é uma citação de outro autor, documento ou fonte. O conteúdo envolvido pela tag.

    Geralmente é exibido com uma formatação especial, como uma indentação ou uma margem adicional em relação ao restante do texto.
    Ela ajuda a fornecer uma formatação visual distintiva para o conteúdo citado, ajudando os leitores a identificar facilmente as partes citadas em um texto.

    Tag Font

    A tag font é uma tag de formatação obsoleta em HTML, não recomendada para uso atualmente. No passado, ela era usada para definir o estilo de fonte de texto, como a cor, tamanho e tipo de fonte. No entanto, o uso direto da tag font é desencorajado por razões de boas práticas e padrões web. Em vez disso, é recomendado usar CSS para aplicar estilos de fonte aos elementos HTML. O CSS oferece mais flexibilidade e controle sobre a formatação visual do texto, permitindo a criação de estilos consistentes e melhores práticas de design.

    Tag Del

    A tag del em HTML é usada para indicar que um trecho de texto foi removido ou deletado em relação à versão anterior do conteúdo. Essa tag é útil em casos em que é necessário mostrar alterações ou revisões em um documento, como em documentos colaborativos ou históricos de revisão. É usada para mostrar o texto deletado ou removido em relação à versão anterior do conteúdo. Ela é útil para exibir alterações ou revisões em um documento.
